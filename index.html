Computational Zeta Functions over Elliptic Curves

I learned the basic ideas behind higher-rank zeta functions for elliptic curves, including introductory background on vector bundles, stability, and counting methods. I also studied the simple result that higher-rank and SLn zeta functions coincide for all n > 1, and wrote small SageMath programs to run basic computations and explore examples.

Hilbert’s Tenth Problem

I studied the statement and significance of Hilbert’s Tenth Problem and what it means for a problem to have an algorithmic solution. I learned the necessary basics in set theory, number theory, and introductory logic and computability—such as Turing machines, incompleteness, and first-order logic—to understand why no algorithm can decide all Diophantine equations.
